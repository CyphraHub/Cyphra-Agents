## What This Is

CYPHRA AI is an AI-driven crypto research and execution layer that helps users analyze tokens, evaluate wallets, and act on insights through a unified multi-platform experience

> [!IMPORTANT]
> Unified account across web app, Telegram Mini App, and browser extension with shared credits and identity

---

## Why It Was Built

Crypto workflows are fragmented and slow

Users jump between tools just to answer:
- is this token safe
- is this wallet smart or risky
- is there new context behind this project

CYPHRA compresses this into a single flow

> [!NOTE]
> Designed as a decision layer first, with a path toward automation later
